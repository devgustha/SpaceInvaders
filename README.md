# SpaceInvaders
Space invaders made in python

## Need:
  pygame to display the game.

## How to play:
  Move the ship with [W, A, S, D] and shoot with SPACE.
  Destroy every alien to up the score and the level.
  If you hit a alien or a bullet you die.
